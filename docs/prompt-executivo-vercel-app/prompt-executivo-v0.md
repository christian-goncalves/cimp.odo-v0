Segue o prompt direto para uso:

---

Atue como um especialista em redesign de landing pages com foco em conversão, UX e reestilização visual sem ruptura estrutural.

CONTEXTO

Você receberá como base uma landing page original (código ou estrutura existente). Existe também uma referência visual de uma primeira refatoração (estilo Falcone), que deve ser usada como inspiração, mas NÃO deve ser replicada.

Seu objetivo é criar uma SEGUNDA VERSÃO da landing page.

OBJETIVO DA LANDING PAGE

A página deve gerar desejo no usuário de melhorar sua autoestima e estética através de implantes dentários.

FOCO:

* apelo emocional
* transformação pessoal
* confiança
* clareza de benefício

CTA PRINCIPAL:

* Agendar consulta

CTA SECUNDÁRIO:

* Ver casos clínicos

PÚBLICO-ALVO

* Classe baixa e média
* Pacientes particulares
* Foco em implante e prótese
* Faixa etária predominante: 50+
* Linguagem: acolhedora, clara, acessível
* Percepção: profissional + confiável + humana
* Evitar excesso de sofisticação elitista

DIREÇÃO VISUAL

Você deve criar uma nova direção visual:

* Inspirada na versão Falcone
* Porém claramente diferente
* Mais comercial
* Mais humana
* Mais direta
* Mais clara no mobile
* Com maior foco em conversão

EVITAR:

* excesso de sofisticação fria
* estética distante
* visual elitizado demais

PERMITIDO:

* reorganizar layout interno das seções
* redesenhar completamente grid, espaçamento e hierarquia
* substituir banners por hero mais refinado
* modernizar totalmente a apresentação visual

RESTRIÇÕES IMPORTANTES

* NÃO alterar a ordem das seções
* NÃO remover seções existentes
* NÃO alterar o conteúdo de forma semântica
* A seção HERO é intocável em estrutura (pode melhorar visual, não mudar essência)

CONTEÚDO

Você pode:

* corrigir gramática
* quebrar textos longos
* transformar textos em bullets
* melhorar títulos
* aumentar escaneabilidade
* resumir trechos redundantes

Você NÃO pode:

* alterar o sentido do conteúdo
* remover informações relevantes

IDENTIDADE VISUAL

* Base nas cores azul e vermelho da versão original
* NÃO copiar a aplicação atual (considerada visualmente fraca)
* Modernizar completamente:

  * suavizar tons
  * melhorar contraste
  * melhorar legibilidade
  * criar estética mais atual

TIPOGRAFIA

* Pode reinterpretar totalmente
* NÃO precisa manter fontes originais
* Priorizar:

  * legibilidade
  * contraste
  * hierarquia clara
  * leitura fácil para público 50+

COMPONENTES PERMITIDOS

Você pode utilizar:

* menu fixo
* hero refinado
* cards modernos
* FAQ (accordion)
* abas
* carrossel (com moderação)
* modal
* before/after (casos clínicos)
* formulário embutido
* botão flutuante de WhatsApp

SEM animações desnecessárias.

RESPONSIVIDADE (PRIORIDADE ALTA)

Mobile é prioridade máxima:

* CTA visível no topo
* leitura fácil
* blocos bem espaçados
* navegação simples
* carregamento rápido
* formulário fácil de usar

SEO E QUALIDADE

Aplicar:

* HTML semântico
* hierarquia correta de headings
* alt em imagens
* boas práticas de acessibilidade
* estrutura otimizada para performance

STACK

Gerar usando:

* Next.js / React
* Componentização por seções
* Estrutura organizada e reutilizável
* Pronto para adaptação posterior

ENTREGA ESPERADA

Você deve entregar:

1. Nova interface completa da landing page
2. Estrutura organizada por seções (componentes)
3. Conteúdo reorganizado (melhor escaneabilidade)
4. Sugestão de assets (onde necessário)
5. Racional de UX (breve por seção)
6. Observações de implementação

IMPORTANTE

* Não copie a versão Falcone
* Não invente estrutura nova
* Trabalhe sobre a base existente
* Evolua o design, não recrie do zero
* Priorize clareza, conversão e acessibilidade
