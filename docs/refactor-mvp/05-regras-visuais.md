# Regras Visuais Minimas

## 1. Coerencia de layout
- Toda secao deve respeitar mesma logica de alinhamento horizontal.
- A transicao entre secoes deve ter ritmo claro, sem blocos "colados".

## 2. Sistema visual
- Manter identidade existente como base.
- Nao criar nova linguagem visual fora do contexto da marca.
- Variar apenas o necessario para ganhar clareza e hierarquia.

## 3. Espacamento
- Usar escala de espacamento consistente em toda a pagina.
- Preservar respiro entre titulo, texto, midia e CTA.
- Evitar acumulacao de elementos sem separacao visual.

## 4. Uso de cores
- Reaproveitar paleta ja utilizada no projeto.
- Usar contraste para guiar leitura e prioridade.
- Evitar excesso de variacoes tonais por secao.

## 5. Comportamento de blocos
- Componentes equivalentes devem ter comportamento equivalente.
- Cards, listas e blocos de destaque devem seguir padrao unico.
- CTA principal deve ter dominancia visual previsivel.

## 6. Responsividade
- Mobile e breakpoint critico de validacao.
- Nenhum bloco pode quebrar leitura, sobrepor ou perder hierarquia.
- Ordem visual precisa continuar clara em telas pequenas.

## 7. Consistencia entre secoes
- Mesmo problema visual recebe mesma solucao visual.
- Evitar "ilhas de design" com padroes contraditorios.
- Pagina final deve parecer um sistema unico, nao colecao de partes.
