# Escopo Visual Consolidado

## Foco da refatoracao
1. Estrutura visual da pagina: distribuicao de secoes, ritmo e leitura por blocos.
2. Hierarquia de informacao: titulos, subtitulos, corpo, CTAs e provas visuais.
3. Composicao das secoes existentes: cards, blocos institucionais, casos, contato e apoio de navegacao.
4. Responsividade: comportamento visual consistente em mobile, tablet e desktop.
5. Consistencia de sistema: tipografia, espacamento, superficies, contraste e repeticao de padroes.
6. Apresentacao final: interface mais clara, confiavel e coesa para MVP visual.

## Limites operacionais
- Nao criar feature nova.
- Nao alterar logica de negocio.
- Nao reestruturar fluxos funcionais.
- Nao expandir escopo para alem da camada visual.

## Saida esperada desta documentacao
- Base unica para o proximo prompt de execucao.
- Encadeamento claro de frentes, etapas e subetapas.
- Criterios objetivos para validar cada etapa visual.
