# Plano de Refatoracao Visual

## Logica de encadeamento
Diagnostico -> frentes visuais -> etapas operacionais -> validacao objetiva.

## Macroblocos de trabalho

### Macrobloco A - Base visual estrutural
- Definir ritmo entre secoes.
- Padronizar grid, largura util e alinhamentos.
- Corrigir hierarquia macro da pagina.

### Macrobloco B - Blocos de entrada e autoridade
- Refinar hero como primeira leitura.
- Reorganizar blocos de prova e credibilidade.
- Ajustar prioridade visual de CTAs existentes.

### Macrobloco C - Composicao de conteudo por secao
- Reequilibrar especialidades, diferenciais, casos e contato.
- Melhorar leitura por bloco sem trocar significado do conteudo.
- Uniformizar comportamento visual entre secoes.

### Macrobloco D - Sistema visual transversal
- Consolidar regras de tipografia, espacamento, cores e superficies.
- Normalizar componentes repetidos.
- Reduzir variacao visual desnecessaria.

### Macrobloco E - Responsividade e fechamento MVP
- Ajustar quebra de layout por breakpoint.
- Garantir consistencia visual ponta a ponta.
- Validar criterios finais definidos no documento de validacao.
